# Packages

- `getassetmanifest` - builds a `manifest.json` file compatible with cloudflare workers
- `llmstxt-generate` - generates a llms.txt file from your `manifest.json`

# Principles

Use alternate convention:

```html
<html>
  <head>
    <link
      rel="alternate"
      type="text/plain"
      href="document.txt"
      title="Plain Text"
    />
  </head>
  <body>
    my site
  </body>
</html>
```
